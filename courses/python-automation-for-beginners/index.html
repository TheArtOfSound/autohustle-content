Installing Python and Required Libraries

Python comes in different versions, but for automation work, you'll want Python 3.8 or newer. Here's how to get started:

Download and Install Python:

  1. Visit python.org and download the latest Python 3.x version
  2. During installation, check "Add Python to PATH" - this is crucial
  3. Verify installation by opening your terminal/command prompt and typing: python --version

Understanding pip:

Python includes pip, a package manager that installs libraries. Think of it as an app store for Python code. You'll use pip constantly in automation projects.

Install Essential Libraries:

Run these commands in your terminal to install libraries you'll use throughout this course:

```bash

pip install requests beautifulsoup4 selenium pandas openpyxl schedule

```

Here's what each library does:

  • requests: Downloads web content and interacts with APIs
  • beautifulsoup4: Extracts data from HTML and XML
  • selenium: Controls web browsers automatically
  • pandas: Handles spreadsheets and data analysis
  • openpyxl: Reads and writes Excel files
  • schedule: Runs tasks at specific times

Setting Up a Code Editor or IDE

You need a good editor to write Python code efficiently. Here are three excellent options:

Option 1: Visual Studio Code (Recommended for beginners)

VS Code is free, lightweight, and has excellent Python support.

  1. Download from code.visualstudio.com
  2. Install the Python extension by Microsoft
  3. Create a new file with .py extension
  4. VS Code will automatically detect Python and offer helpful features

Option 2: PyCharm Community Edition

More feature-rich but heavier. Great for larger projects.

  1. Download from jetbrains.com/pycharm
  2. Choose Community Edition (free)
  3. Create a new Python project to test it works

Option 3: IDLE (Comes with Python)

Basic but functional. Good for quick scripts and learning.

Essential Editor Features to Learn:

  • Syntax highlighting: Colors that make code easier to read
  • Auto-completion: Editor suggests code as you type
  • Debugging: Step through code line-by-line to find issues
  • Terminal integration: Run Python scripts without leaving the editor

Test Your Setup:

Create a file called test.py and write:

```python

print("Hello, automation world!")

import requests

print("All libraries imported successfully!")

```

Run this file. If it works without errors, your environment is ready.

Action Items:

  • [ ] Choose and install one of the three editors

  • [ ] Create and run the test.py file successfully

Understanding the Importance of Version Control

Version control tracks changes to your code over time. It's like having unlimited "undo" for your entire project, plus the ability to collaborate with others.

Why You Need Version Control:

  • Backup: Never lose your work again
  • History: See what changed and when
  • Experimentation: Try new approaches without fear
  • Collaboration: Work with others on the same code
  • Rollback: Return to working versions when things break

Git Basics:

Git is the most popular version control system. Here's what you need to know:

Install Git:

  1. Download from git-scm.com
  2. Use default settings during installation
  3. Verify with git --version in your terminal

Essential Git Commands:

  • git init: Start tracking a folder
  • git add filename.py: Stage files for saving
  • git commit -m "message": Save changes with a description
  • git status: See what's changed
  • git log: View history of changes

GitHub for Backup:

GitHub stores your code online for free. Create an account at github.com and learn to push your local projects there.

Practical Exercise:

  1. Create a new folder called python-automation
  2. Open terminal in that folder
  3. Run git init
  4. Create a simple Python file
  5. Run git add . then git commit -m "Initial commit"
  6. You've just created your first repository!

Action Items:

  • [ ] Install Git on your computer
  • [ ] Create a GitHub account
  • [ ] Complete the practical exercise above
  • [ ] Commit your test.py file to your new repository
  • [ ] Practice the five essential Git commands
