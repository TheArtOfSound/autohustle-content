Lesson 1.1: Understanding Data Types in Excel

Data types refer to the categories of data that can be stored in a cell. Understanding data types is essential for accurate calculations and data manipulation.

In Excel, there are several built-in data types:

  • Number: used for numerical values (e.g., 123, -0.5)
  • Text: used for strings or characters (e.g., "hello", "world")
  • Date: used for dates and times
  • Boolean: used for true/false values (e.g., TRUE, FALSE)
  • Error: used to indicate errors or missing data
