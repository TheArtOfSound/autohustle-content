Setting Up Your Brand Voice

Before diving into the templates, establish your foundation with these key considerations:

Voice vs. Tone

  • Voice: Your brand's consistent personality (friendly, authoritative, quirky)
  • Tone: How voice adapts to context (casual social posts vs. formal whitepapers)

Essential Voice Elements to Document

1. Personality traits (3-5 key characteristics)

2. Communication style (formal/informal, technical/accessible)

3. Vocabulary preferences (industry jargon, brand-specific terms)

4. Sentence structure (short and punchy vs. complex and detailed)

5. Emotional approach (optimistic, pragmatic, empathetic)

Template 2: AI Prompt Engineering for Voice Consistency

Purpose: Create standardized prompts that embed your brand voice into AI-generated content

Master Voice Prompt Template

```

You are writing as [BRAND NAME], a [BRAND DESCRIPTION] that serves [TARGET AUDIENCE].

VOICE CHARACTERISTICS:

  • Personality: [LIST 3-5 KEY TRAITS]
  • Tone: [FORMAL/CASUAL/PROFESSIONAL/CONVERSATIONAL]
  • Style: [SENTENCE STRUCTURE PREFERENCE]

LANGUAGE GUIDELINES:

  • Use: [PREFERRED WORDS/PHRASES]
  • Avoid: [WORDS/PHRASES TO NEVER USE]
  • Technical Level: [JARGON PREFERENCE]

WRITING RULES:

1. [SPECIFIC RULE ABOUT SENTENCE LENGTH]

2. [RULE ABOUT ADDRESSING AUDIENCE]

3. [RULE ABOUT CALL-TO-ACTIONS]

4. [RULE ABOUT EMOTIONAL TONE]

5. [ADDITIONAL BRAND-SPECIFIC RULE]

Now write [CONTENT TYPE] about [TOPIC] following these voice guidelines.

```

Content-Specific Prompt Variations

Blog Posts:

```

Following the [BRAND NAME] voice profile above, write a [WORD COUNT] blog post about [TOPIC] that:

  • Starts with [PREFERRED OPENING STYLE]
  • Includes [NUMBER] subheadings
  • Ends with [PREFERRED CTA STYLE]
  • Maintains [TONE] throughout

```

Social Media:

```

Create [NUMBER] social media posts for [PLATFORM] about [TOPIC] using [BRAND NAME]'s voice:

  • Keep posts under [CHARACTER LIMIT]
  • Use [HASHTAG STRATEGY]
  • Include [EMOJI PREFERENCE]
  • Match [PLATFORM-SPECIFIC TONE]

```

Email Marketing:

```

Write a [EMAIL TYPE] email for [BRAND NAME] about [TOPIC] that:

  • Subject line reflects [BRAND VOICE TRAIT]
  • Opening matches [GREETING STYLE]
  • Body maintains [CONVERSATIONAL LEVEL]
```
