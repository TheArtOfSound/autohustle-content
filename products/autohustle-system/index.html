What AutoHustle Does

AutoHustle is a fully autonomous income generation system that runs 24/7. It orchestrates 15 revenue strategies simultaneously:

  • SEO Content — Generates and publishes 100+ articles/day targeting buyer-intent keywords with affiliate links
  • Shopify Store — Runs a complete e-commerce store: sources products from CJ Dropshipping, writes descriptions, manages collections, auto-fulfills orders
  • Crypto Trading — HFT scalping mode: buys top momentum movers, holds 3 minutes, repeats every 5 minutes
  • Digital Products — Creates and lists templates, guides, and prompt packs on Stripe
  • Micro-SaaS Tools — Deploys 10 web utility tools to GitHub Pages with premium tiers
  • Email Outreach — B2B lead discovery and automated email campaigns
  • Course Creator — Generates and lists online courses on Gumroad
  • Marketing Engine — Runs email campaigns, social posts, and blog content for the Shopify store

System Requirements

  • Python 3.11+ (Windows, Mac, or Linux)
  • 4GB+ RAM recommended (8GB for running Ollama locally)
  • Git installed and configured
  • Internet connection (for API calls)

Quick Start (15 Minutes)

Step 1: Install Python Dependencies

git clone https://github.com/theartofsound/autohustle.git
cd autohustle
pip install -r requirements.txt

Step 2: Configure Your API Keys

Run the interactive setup wizard:

python main.py --setup

You'll be prompted for each API key. Start with these free ones:

  • GitHub Token — For publishing to GitHub Pages (free)
  • Groq API Key — For AI content generation (free 500K/day)
  • Gemini API Key — Backup AI provider (free 1K/day)

Step 3: Set Up GitHub Pages

  1. Create two GitHub repos: autohustle-content and autohustle-tools
  2. Enable GitHub Pages in each repo's Settings → Pages → Deploy from main branch
  3. Add your GitHub token + username in AutoHustle settings

Step 4: Start AutoHustle

# Start main agent (runs 24/7)
python main.py

# Start with dashboard (localhost:8888)
python main.py --dashboard

# Run one cycle then exit (for testing)
python main.py --once

Configuration Guide

All settings are in config.yaml. Key sections:

brand:
  name: "YourBrand"
  domain: "yourdomain.com"
  github_username: "yourusername"
  content_repo: "your-content-repo"
  tools_repo: "your-tools-repo"

strategies:
  shopify_store:
    enabled: true
    store_name: "Your Store"
    price_min: 15
    price_max: 79

  crypto_trading:
    enabled: true
    max_daily_loss_usd: 50  # Safety limit

Safety Systems

  • Kill Switch: Create file data/.kill_switch to immediately halt all activity
  • Trading Guard: Auto-halts crypto trading if daily loss exceeds $50
  • Circuit Breaker: Auto-pauses failing strategies after 5 consecutive errors
  • Budget Guard: Hard cap on daily spending ($0 by default — free tier only)

Monitoring Your Progress

  • Dashboard: Run python main.py --dashboard → open localhost:8888
  • Logs: Check logs/autohustle-YYYY-MM-DD.log
  • Database: Query data/autohustle.db with any SQLite client
