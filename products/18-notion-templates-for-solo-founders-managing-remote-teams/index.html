| T-005 | [e.g., Onboard new contractor] | [Your Name] | [Operations] | [MM/DD] | ⚪ Not Started | Low | Awaiting contract approval |

🔑 Legend

  • Status:
  • 🟢 Done
  • 🟡 In Progress
  • 🔴 Blocked
  • ⚪ Not Started
  • Priority:
  • Critical (immediate impact)
  • High (must complete this week)
  • Medium (next week OK)
  • Low (optional/nice-to-have)

🔄 Weekly Review Process

  1. Every [Day, e.g., Friday], review all tasks with status ≠ 🟢
  2. Update: Due dates, notes, ownership if needed
  3. Unblock: Resolve dependencies during team check-in
  4. Archive completed tasks monthly

📣 Delegation Best Practices

  • Assign ONE owner per task
  • Set clear due dates (not just “ASAP”)
  • Link to related docs (e.g., brief, design, ticket)
  • Use @mentions in comments for questions

📌 Sync this with your project tool. Use as single source of truth.

```
