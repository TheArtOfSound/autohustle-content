Brand Consistency Guidelines

#### Color Usage Rules

  • Primary color: 60% of design elements
  • Secondary color: 30% of design elements
  • Accent color: 10% for highlights and calls-to-action
  • Always maintain sufficient contrast for readability

#### Typography Hierarchy

  • H1: Primary font, 24-36pt, primary color
  • H2: Primary or secondary font, 18-24pt, secondary color
  • Body: Secondary font, 12-16pt, dark neutral
  • Captions: Secondary font, 10-12pt, medium neutral

#### Image Treatment

  • Apply subtle sepia or vintage filters
  • Use consistent border styles across all images
  • Maintain aspect ratios that complement template layouts
  • Ensure images align with your chosen era's aesthetic

Format Notes

File Formats and Specifications

#### Vector Files (AI, EPS, SVG)

  • Resolution: Scalable to any size without quality loss
  • Best For: Logos, icons, and graphics requiring resizing
  • Editing: Requires vector editing software (Adobe Illustrator, Inkscape)
  • Export Options: Can be converted to raster formats as needed

#### Raster Files (PSD, PNG, JPG)

  • Resolution: 300 DPI for print, 72 DPI for web
  • Best For: Detailed graphics, photographs, and complex layouts
  • File Sizes: Larger files for higher quality, optimize for intended use
  • Transparency: PNG files support transparent backgrounds

#### Document Templates (INDD, DOCX, PPTX)

  • Layout: Pre-formatted with vintage styling
  • Text: Editable with proper font specifications
  • Images: Placeholder boxes for easy replacement
  • Export: Multiple format options for different distribution needs

Color Specifications

#### Digital Use (RGB)

All digital templates use RGB color mode for optimal screen display:

  • Web Graphics: sRGB color space recommended
  • Presentations: RGB ensures vibrant colors on projectors

#### Print Use (CMYK)
