Crafting Compelling Content

Hook & Headlines (Prompts 1-8)

Prompt 1: The Scroll-Stopper

```

Create 10 attention-grabbing headlines for [your topic/industry] that make people stop scrolling immediately. Each headline should:

  • Start with a power word (Urgent, Secret, Proven, etc.)
  • Include a specific number or timeframe
  • Promise a clear benefit or outcome
  • Be under 60 characters for social media optimization

Target audience: [describe your audience]

```

Prompt 2: The Curiosity Gap Creator

```

Generate 5 headlines that create an irresistible curiosity gap about [your subject]. Each should:

  • Hint at valuable information without revealing it
  • Use phrases like "The one thing..." or "What nobody tells you about..."
  • Include emotional triggers (fear, excitement, surprise)
  • End with a compelling reason to click

Format as: [Intriguing statement] + [Benefit/consequence]

```

Prompt 3: The Problem-Agitator

```

Write 7 headlines that agitate a specific problem your audience faces regarding [your niche]. Structure each as:

  • Identify the pain point clearly
  • Amplify the frustration or consequence
  • Hint at the solution without giving it away
  • Use urgent language that compels immediate action

Problem focus: [specific problem your audience faces]

```

Prompt 4: The Social Proof Amplifier

```

Create 6 headlines incorporating social proof elements for [your product/service/content]. Include:

  • Specific numbers (customers served, results achieved, etc.)
  • Testimonial snippets or success stories
  • Industry recognition or awards
  • Community size or engagement metrics

Social proof data: [your specific metrics/testimonials]

```

Prompt 5: The Contrarian Take

```

Generate 8 contrarian headlines that challenge common beliefs in [your industry]. Each should:

  • Start with "Why everyone is wrong about..."
  • Challenge a popular myth or assumption
  • Promise to reveal the "real truth"
  • Position you as the knowledgeable insider

Common belief to challenge: [specific industry assumption]

```

Prompt 6: The Time-Sensitive Urgency

```

Write 5 headlines that create genuine urgency around [your offer/content] without being spammy:

  • Reference limited time or availability
  • Mention seasonal relevance or trends
  • Include countdown elements
  • Connect to current events or deadlines

Urgency factor: [your specific time constraint]

```

Prompt 7: The Benefit Stack

```

Create 4 headlines that stack multiple benefits for [your target audience]:

  • Lead with the primary benefit
  • Add 2-3 secondary benefits
  • Use "plus" or "and" connectors effectively
  • Keep total length under 80 characters

Primary benefit: [main value proposition]

Secondary benefits: [additional value points]

```

Prompt 8: The Question Hook

```

Generate 10 question-based headlines for [your content topic] that:

  • Start with "What if..." "How would..." or "Why do..."
  • Address your audience's deepest desires or fears
  • Imply that you have the answer
  • Create mental engagement before they even click

Audience desire/fear: [specific emotional trigger]
```
