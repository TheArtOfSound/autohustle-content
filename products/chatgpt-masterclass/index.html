Module 1: Prompt Engineering Fundamentals

Module 1: Prompt Engineering Fundamentals

The CRISP Framework

Every high-quality prompt follows five principles — Context, Role, Instruction, Specifics, Parameters:

  1. Context — Give background the AI needs to understand your situation
  2. Role — Tell the AI who it should be (expert copywriter, analyst, coach)
  3. Instruction — State exactly what you want done
  4. Specifics — Include details like word count, tone, audience, format
  5. Parameters — Set constraints (avoid jargon, use bullet points, include examples)
